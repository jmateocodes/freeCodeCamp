All jQuery selectors start with a '$' You can target elements by type $("button"), class $(".well"), or by ID $("#target3") You can add a class using jQuery .addClass(), and remove one with .removeClass() You can use the .css() function in jQuery to change the CSS of an element jQuery's .prop() function allows you to change the proporties of elements jQuery has a function called .html() that lets you add HTML tags and text within an element. jQuery has a .text() function that only alters texts, not tags .remove() removes an HTML element entirely.
